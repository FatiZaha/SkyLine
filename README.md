## ğŸ“ Architecture
SkyLine utilizes a **Decoupled Full-Stack Architecture**:
- **UI/UX Layer:** Built with **Tailwind CSS** for rapid layout design and **MUI** for complex, accessible interactive components.
- **Service Layer:** A **Spring Boot** API ensures high availability and handles the heavy lifting of business logic.
- **Security Layer:** Stateless authentication via **JWT** ensures a secure bridge between the React client and the Spring server.

## âš™ï¸ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/FatiZaha/SkyLine.git
```

### 2. Backend Setup (Spring Boot)
- Update `src/main/resources/application.properties` with your MySQL server details.
- Build and run the project using Maven or your preferred IDE.

```bash

```
- The application will be live at `http://localhost:3000`.
